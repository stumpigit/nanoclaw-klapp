# Klapp School Messages

When the user asks about Klapp messages, use the `mcp__nanoclaw__klapp_read_messages` tool. No credentials needed — they are configured on the host.

## Usage

Call `mcp__nanoclaw__klapp_read_messages` whenever the user asks about:
- Klapp Nachrichten / messages
- School messages / Schulnachrichten
- "Was steht in Klapp?" / "check klapp"
